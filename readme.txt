版本 version 1.0.2 2015/09/21
1 输入一个样点，程序计算出此样点和其他所有待推测点的相似度和不确定性；
2 结果生成一个相似度图层和一个不确定性图层；
3 程序在控制台中能够显示出和输入样点相似度一致的样点以及最相似的几个样点（图片20150921201505.jpg）；
版本 version 1.0.3 2015/09/24
1 程序将与样点相似度差值小于0.02的待推测点作为可供选择的可替代点（即候选样点）；
2 程序显示出可替代点的地理坐标位置；
版本 version 1.0.4 2015/10/09
1 此版本更新了程序组织形式，将main函数里面获取参数的一大波语句放在了新建的BasicParameter.cpp中；
2 添加了距离计算的功能，用户可以选择100（此数值后期应由用户设定）米范围内的相似点；
版本 version 1.0.5 2015/10/13
1 增加了NeighborSimi.cpp；
2 增加了计算输入样点周围一定邻域内所有栅格的相似度值的平均值；
3 增加了计算每一个候选样点周围一定邻域内所有栅格的相似度值的平均值；
版本 version 1.0.6 2015/10/16
1 删除了计算输入样点周围一定邻域内所有栅格的相似度值的平均值的功能；
2 增加了计算每一个候选样点周围一定邻域内所有栅格的相似度值的方差；
3 对所有候选样点周围邻域相似度值的平均值和方差进行排序：先按照平均值的大小排序，平均值大的候选样点优先选择，若遇到平均值相等的样点按照他们的方差从小到大将这几个样点进行排序；
